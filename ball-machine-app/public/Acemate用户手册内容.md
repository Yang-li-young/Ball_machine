

# 快速开始

> 了解如何快速安装和启动您的ACEMATE机器人，请勿在非标准场地使用Rally模式，非标准场地请使用BallMachine模式
# 设备使用指南

## 开关机

- 正常按下开关按钮即可开/关机，开机过程中尽量保持机器静止，机器开机过程会红灯闪烁，机器完成自检后指示灯变为绿/紫色常亮，表明机器正常开机，即可使用APP连接设备蓝牙开始使用。。。。。
## 安装接球网

- 在伸缩接球网过程中，请勿直接将某一根伸缩杆直接伸缩到最顶端，需将四个接球伸缩杆依次伸缩至顶端；
- 注意伸缩过程中不要让球网钩住内部的拨杆和螺杆
##  安全检查

- 使用ACEMATE前请确保以下内容：
	- 电池电量充足；
	- 周围环境安全；
	- 场地平整、无障碍物；
	- 检查设备各部件是否完好，确认发球口无堵塞、轮子正常行走；
## 电池充电

- 可直接使用充电器插入机器人开关下方充电口进行充电
- 也可将电池从电池仓取出充电

# APP页面说明

## Rally模式页面说明

### 状态显示栏

- 左上角为蓝牙连接设备状态，开机后连接Acemate_XXXX设备即可控制；
- 右上角为Acemate视觉识别状态，当机器识别到位置正确后会由“Incorrect Positon”变为“Correct Position”。
### 球场展示区域
- 依次点击球场落点编号设置不同的落点顺序，机器落点的准确性和网球、场地等类型相关性较高，请尽量保证网球的一致性和在标准网球场使用。
### 上下旋和高度展示区域
- 可视化机器击球的上下旋和过网高度，可通过速度、上下旋滑杆进行调整。
### 球组设置区域
- 选择球组的上下旋和球速
### 小车设置卡片
- Car Speed设置小车的速度（由于场地材质的差异，如果机器发射球的左右偏差较大，可适当降低车速）
- Machine Vertical Position用于调整机器在球场纵向位置，可根据不同的练习需求，结合落点设置，实现小场对拉（1，2号Position），底线对拉（4、5号position）。
- Machine Horizontal Positon用于调整机器在球场纵向位置，可根据不同的练习需求，结合落点设置，实现不同线路组合的练习。

### 预设训练方案

- 支持用户选择预设的训练方案

# APP功能使用建议

> 针对不同水平的网球爱好者，我们建议使用机器相关的设置方法

## **NTRP 1.0–2.0：入门者阶段**

> 目标：建立击球感觉与稳定性，找到“网球的节奏”

**推荐使用模式：** 发球机模式（固定点出球）

**训练方法：**
1. 将机器放置在底线中点位置，使用预设方案的基础定点训练线路，从“左右两侧浅球定点练习”依次进阶至“左右两侧深球定点练习”，练习正反手稳定性，控制击球落点和节奏。
2. 进一步使用预设训练方案的基础双线训练线路，切换正反手。
3. 结合App视频回放功能，观察是否始终保持一致的挥拍节奏。

**APP设置方法：**
- 1. 在Ball Machine页面点击预设训练方案，选择相应的方案
- 2. 可以依据自己的需求调整出球速度
- 3. 调整俯仰角度控制过网高度
- 4. 调整上旋/平击/下旋等击球类型
- 5. 调整发球时间间隔改变击球节奏

## **NTRP 2.0–3.0：基础稳定阶段**

> 目标：形成主动击球意识与方向控制能力

**推荐使用模式：** 发球机模式 + Rally模式单点训练

**训练方法：**

1. 使用发球机模式，将机器放置在底线中点位置，使用预设方案的深浅球组合线路和进阶线路。
2. 使用Rally模式，将机器放置在中场靠近底线位置 **（推荐设置Machine Vertical Position 1/2/3，Machine Horizontal Position 5）**，选择浅落点，设置较低速度，和机器进行小场对拉.

**APP设置方法：**

- 1. 在 **Ball Machine 页面 → 预设方案 →深浅球组合线路/进阶线路**，结合难度和自身需求调整发球方向、速度、旋转、俯仰角度、发球时间间隔等参数
- 2. 在**Rally页面选择单个中路浅落点（3/8/13），调整适合自己的速度，设置Machine Vertical Position 1/2/3，Machine Horizontal Position 5）** 控制击球落点，实现稳定对拉

## **NTRP 3.5–4.0：稳定控制阶段**

> 目标：提升旋转掌控与击球质量，形成连续对打节奏

**推荐使用模式：** Rally模式

**训练方法：**

1. 使用 **Rally 模式**，将机器放置在底线中点位置（设置Machine Vertical Position 4/5，Machine Horizontal Position 5），选择点位中3–5个点，进行连续对打训练。
2. 使用 **Rally 模式**，将机器放置在底线右侧位置（设置Machine Vertical Position 4/5，Machine Horizontal Position 1/2/3），选择点位中3–5个点，进行连续对打训练。

**APP设置方法：**

1. 打开 **Rally 模式 → 选择训练点位（3–5个）、设置Machine Vertical Position 4/5，Machine Horizontal Position 5**；
2. 调整适当的速度和旋转。

# 常见问题

## 硬件问题

- 接球网可以接住球但弹出较多
	- 请将接球网伸缩至合适的高度，推荐高度为伸缩杆的最高。
- 转盘堵转
	- 可能的现象是开启后机器不发球，请在发球机模式下观察机器，确认是否是储球转盘未转动。
- 发球电机堵转
	- 可能的现象是开启后机器不发球，请在发球机模式下观察机器，确保储球转盘没有球后观察发球电机是否转动。
	- 查看是否有球卡在上下发球电机中间，如有球，请关闭机器，拆卸机器主体两侧螺丝，提起转球装置取出网球即可。
- 车轮不转动
	- 确保机器处于正常开机状态（指示灯为绿色或蓝色），手动控制机器行走。
## 视觉问题

- 机器人在运行过程中可能碰到以下视觉相关的问题
	- Rally模式下正常使用过程中突然停止，可能是球场标识线（球场发球线、边线、球网重力带缺失等）不明显导致机器丢失位置，手动移动至正常位置即可
	- Rally模式下接球效果不佳，观察球场环境中是否存在类似网球（黄色、圆形）的图像或物体，请进行移除或者遮挡
	- 机器突然移动向某个方向，可能是有网球或网球类似物在机器人视线中移动，请避免此类情况
